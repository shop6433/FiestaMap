v1.0.4 추가사항(200302)
첫 시작 화면 후 추가하기 버튼을 누르면 팝업 창 뜨고, 버튼이 추가 되도록 해놓은 상태
사용한 방법
- Framelayout위에 기존 mapfragment랑 겹쳐서 linearlayout 같이 올려두고 최초 set INVISIBLE상태에서 클릭 시 VISIBLE로 전환 되도록 했음
예상되는 문제점
- INVISIBLE 화면 클릭시에 버튼들이 비 활성화 되는지, 모양만 보이지 않는 것인지 잘 모르겠음.
